1.INTRODUÇÃO AO HTML
HTML (Hypertext Markup Language ou Linguagem de Marcação de Hipertexto) é uma linguagem utilizada para construção de páginas de Internet. HTML define um conjunto de regras que permite a formatação de páginas. Um documento HTML consiste em um arquivo de texto que será lido e interpretado como uma página por browsers (navegadores).
